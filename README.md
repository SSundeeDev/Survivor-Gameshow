# Survivor: The CapEx Budget

Interactive gameshow tool for Track Record Assets.

## Deploy to Vercel (one-time setup)

1. Upload this entire folder to a new GitHub repository
2. Go to vercel.com, sign in with GitHub
3. Click "Add New... → Project"
4. Select this repo and click Deploy
5. Vercel auto-detects Vite and deploys in ~60 seconds
6. Your live URL will be something like `capex-game.vercel.app`

## Updating the app

1. Replace `src/App.jsx` with the new version
2. Commit the change in GitHub (web editor works fine)
3. Vercel auto-redeploys in under a minute
